# Nest Plan 📅✨

Nest Plan is a mobile productivity and scheduling application built with **Flutter** and **SQLite**. It allows users to manage daily tasks, organize long-term projects with date ranges, and automate repetitive routines using a custom template system.

## 🚀 Features

- **Local-first Architecture:** Full data control using SQLite.
- **Flexible Planning:** Create daily tasks or define multi-day date ranges.
- **Smart Templates:** Save your routines and apply them to specific dates in one click.
- **Sub-task Management:** Organize complex projects with nested tasks.
- **Clean UI/UX:** User-friendly design with light/dark mode support.
- **Data Security:** Secure local storage with confirm-dismiss protocols.

## 📦 Technologies

- **Flutter** (Dart)
- **SQLite** (sqflite)
- **State Management:** Provider/setState
- **UI Components:** Lucide Icons, TableCalendar, Intl

## 📱 Platforms

Developed and tested for **Android**.

## 📌 Goal

This project was developed for the **EFC304 Mobile Application Design & Development** course. It aims to demonstrate robust local database integration and efficient state management in mobile development.

## ⚙️ Setup Instructions

1. Clone the repository:

- git clone https://github.com/olmezarda/nestplan.git
- cd repository

2. Install dependencies:

- flutter pub get
  
3. Run the app on an Android device or emulator:

- flutter run
